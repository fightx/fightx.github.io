如何设置共享文件夹：

Windows xp的共享分为两种：简单文件共享（Simple File Sharing）和高级文件共享（Professional File Sharing）。xp在默认情况下是打开简单文件共享的。

1. 简单文件共享

(1.“使用简单共享”   2.“右击设置要简单共享的驱动器或文件夹” 3.打开XP的GUEST账户 4.如果仍不共享,请运行GPEDIT.MSC打开”组策略”….   从“拒绝从网络访问这台计算机”中删除guest帐户 5.打开防火墙中相应设置 )

示例：销售部经理秦天SEO想将D盘根目录下gx文件夹共享给其他同事使用，便于秦天SEO及时了解销售情况，他采取了很多方式，过程中也遇到了很多问题，在不断的努力学习下，终于克服了种种困难，让我们一起看看，他是怎么实现共享的！

1.3 在图1-4中，我们只需勾选在网络上共享这个文件夹就可以，共享名就是他人访问你资源时显示的文件夹名，可以与原文件夹名不同，单击确定就可以完成简单的共享！

1.4 启用GUEST（来宾）账户

2. 高级文件共享（xp家庭版没有高级共享）

XP的高级文件共享是通过设置不同的帐户，分别给于不用的权限，即设置ACL（Access Control List，访问控制列表）来规划文件夹和硬盘分区的共享情况达到限制用户访问的目的。高级文件共享更加安全，更加人性化，所以会详细的讲解。

注意:为了更好的使用高级文件共享的资源，建议将 FAT 卷转换成 NTFS。

CONVERT volume /FS:NTFS [/V] [/CvtArea:filename] [/NoSecurity] [/X]

volume      指定驱动器号(后面跟一个冒号)、

装载点或卷名。

/FS:NTFS    指定要被转换成 NTFS 的卷。

/V          指定 Convert 应该用详述模式运行。

/CvtArea:filename

将根目录中的一个接续文件指定为

              NTFS 系统文件的占位符。

/NoSecurity 指定每个人都可以访问转换的文件

和目录的安全设置。

/X          如果必要，先强行卸载卷。

该卷的所有打开的句柄则无效。

2.1 首先，关闭简单文件共享。打开我的电脑，工具——文件夹选项，取消如图2-1中的使用简单文件共享的勾即可,也可以在组策略里面进行设置.具体可以参照Win7网络共享本地安全策略设置,将网络访问: 本地帐户的共享和安全模型 改为“经典–本地用户以自己的身份验证”

                         图2- 1

2.2 接下来的步骤同简单文件共享的1、2两步，不过此时出现的对话框有点不同，如图2- 2多出了权限等按钮，单击权限按钮，出现图2-3中的共享权限设置对话框，这里我们可以给用户或组指派访问共享文件夹的权限，一般我们只需给用户Everyone设置读取权限即可，秦天SEO为了给销售部的其他同事可以及时的将销售记录发到自己电脑上，于是秦天SEO创建了saler用户，并在共享权限里面添加并给予完全控制权限。
