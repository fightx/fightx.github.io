快速创建任意大小文件的方法

Written by 秦天SEO with ♥ on 2011-10-19 in 操作系统 , 558 words

有的时候秦天SEO要测试U盘速度，手头又没有什么大文件，如何凭空的创建出4G的、6G、10G、100G、1TB的文件？难道在记事本里面打字、复制、粘贴？那有没有什么方法，快速的创建指定大小的文件呢？答案是肯定的。

DOS命令之fsutil

这里要用到的就是DOS命令里面的一个命令：fsutil ，通过这个命令可以快速的创建任意大小的文本文件，使用方法如下：

E:>fsutil file createnew 1.txt 10485760 //这里的10485760为10G
已创建文件 E:\1.txt

ps:1G=1024*1024*1024=1048576

fsutil的功能简介

E:>fsutil

—- 支持的命令 —-

8dot3name       8dot3name 管理

behavior        控制文件系统行为

dirty           管理卷的已损坏位数

file            文件特定命令

fsinfo          文件系统信息

hardlink        硬链接管理

objectid        对象 ID 管理

quota           配额管理

repair          自疗管理

reparsepoint    重分析点管理

resource        事务资源管理器管理

sparse          稀疏文件控制

transaction     事务管理

usn             USN 管理

volume          卷管理

E:>fsutil file createnew

用法 : fsutil file createnew

例如 : fsutil file createnew C:\testfile.txt 1000

E:>fsutil file createnew 1.txt 10485760

已创建文件 E:\1.txt

fsutil的妙用

有U盘或者移动硬盘的朋友,喜欢把重要资料放在里面,当有人向你借U盘或者移动硬盘去网吧拷电影,而这里面有很多重要资料,一旦中毒,后果不堪设想.而你不借又不近人情,这个时候就可以创建一个和硬盘剩余空间同样大的文件，就可以了。

还有，孩子放假了，希望下载游戏玩，结合重要资料”藏”起来之系统篇,创建删除不掉的文件夹，并用fsutil创建大的文本文件放入其中，就可以实现硬盘的“写保护”。
