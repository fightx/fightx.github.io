用友ERP U890+SQL Server 2005安装全过程及注意事项

Written by 秦天SEO with ♥ on 2010-12-22 in 网络维护 , 879 words

安装步骤很简单：

1.      修改计算机名称，最好是英文和数字的组成，不含有“-”

2.      先安装SQL Server 2005，SQLServer2005SP2简体中文企业完整版（1.2G）下载地址：http://www.onegreen.net/Soft/HTML/20460.html

安装过程中，记得指定sa账户，设置密码

3.      安装iis5.1，下载地址：http://www.orsoon.com/soft/7748.html

4.    如何手动重新生成性能计数器库：http://support.microsoft.com/kb/300956

  1. 文中，我只是将启动注册表编辑器，然后在注册表中查找以下项：

HKEY_LOCAL_MACHINE\Software\Microsoft\Windows NT\CurrentVersion\Perflib

  1. 在注册表中，将“LastCounter”值更改为 1846（十进制），并将“LastHelp”值更改为1847（十进制）。

修改后就可以了，如果不行，请参考文中其他方法

 5.      升级SQL Server 2005到sp3，下载地址：http://www.microsoft.com/downloads/details.aspx?displaylang=zh-cn&FamilyID=ae7387c3-348c-4faa-8ae5-949fdfbe59c4

6.      安装ERP U890。.根据提示缺少的文件：点击提示处，打开到安装目录/\3rdProgram，然后安装相应组件就可以了

 

7.      至此安装基本完成，如果还有什么问题，本人随时更新！
